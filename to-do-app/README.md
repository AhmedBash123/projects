# To-Do List App

A simple and responsive **To-Do List application** built with **HTML, CSS, and JavaScript**.  
This project was created to practice JavaScript fundamentals and DOM manipulation.

## Features

- Add and remove tasks
- Mark tasks as completed
- Empty state illustration when no tasks exist
- Clean glassmorphism UI
- Responsive design
- Uses Local Storage to save tasks

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome

## What I Learned

- DOM manipulation
- Event handling in JavaScript
- Working with Local Storage
- CSS positioning and layouts
- Building small interactive projects

## How to Run the Project

1. Clone or download the repository
2. Open `index.html` in your browser
3. Start adding tasks

## Purpose

This project is part of my JavaScript learning journey and was built to strengthen my understanding of front-end development basics.
