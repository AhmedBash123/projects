# Currency Converter

This is a simple currency converter built with JavaScript.

## What it does
- Converts between different currencies
- Uses live exchange rates
- Updates the converted value automatically

## Built with
- HTML
- CSS
- JavaScript
- Exchange Rate API

## How to use
1. Get a free API key from [ExchangeRate-API](https://www.exchangerate-api.com/).  
2. Open `script.js` in the project folder.  
3. Replace `"YOUR_API_KEY_HERE"` with your API key.  
4. Open `index.html` in your browser.  
5. Select currencies and enter the amount to see the conversion.

## Notes
- The API key is required to fetch live rates.  
- This project is for learning purposes.
