# Dragon Ball Anime Generator

This is a small web project that fetches Dragon Ball characters from a public API and displays a **random character** with their **image and name**.  

Every time you click the "Get Anime" button, a new character is shown dynamically.

---

## Features

- Fetches **all Dragon Ball characters** using the Dragon Ball API.
- Picks a **random character** from all available pages.
- Shows **character image and name**.
- Button is disabled while loading to avoid multiple requests.
- Container shows only after the image is loaded.

---

## Skills Gained

- **JavaScript (async/await & fetch API):** Learned how to fetch data from a REST API and handle JSON.  
- **DOM Manipulation:** Learned to dynamically update HTML elements like images, text, and buttons.  
- **Handling API pagination:** Learned how to fetch all pages of data from an API.  
- **Basic UI interactivity:** Learned how to show/hide elements and use loading states.

---

## How to Run

1. Open `index.html` in your browser.  
2. Click the "Get Anime" button to see a random Dragon Ball character.  

---

## API Used

- [Dragon Ball API](https://www.dragonball-api.com/) - Public API for Dragon Ball characters and planets.
